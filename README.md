# Zomato Restaurant EDA — Exploratory Data Analysis

## Project Overview
Exploratory Data Analysis on the Zomato Restaurants Dataset (India) as part of the
Pluto Academy AI & Machine Learning Internship Program.

## Dataset
- **Source:** [Zomato Restaurants Dataset — Kaggle](https://www.kaggle.com/datasets/shrutimehta/zomato-restaurants-data)
- **Records:** 9,551 restaurants (8,652 India-specific after filtering)
- **Features:** 21 columns including city, cuisines, ratings, cost, delivery options

## Tools & Libraries
- Python, Pandas, NumPy, Matplotlib, Seaborn
- Google Colab

## Project Structure
- Data Loading & Inspection
- Data Cleaning (missing values, irrelevant columns, India filter)
- Exploratory Data Analysis (5 business questions)
- Visualizations (6 charts)
- Insights Report (5 data-backed business insights)

## Key Insights
1. New Delhi dominates with 5,473 restaurants — 5x more than any other city
2. North Indian cuisine is the most listed — fusion menus are highly popular
3. 72% of restaurants still don't offer online delivery — massive growth gap
4. Higher price range = higher average rating across all restaurants
5. Most restaurants are either unrated or rated between 3.0–4.5
